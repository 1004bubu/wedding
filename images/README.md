# 이미지 폴더

사진 파일을 여기에 넣으세요. 파일이 없으면 청첩장에는 자동 placeholder가 표시됩니다.

## 구조

```
images/
├─ cover/
│  └─ main.jpg        ← 대문(메인) 사진 1장. 권장 비율 3:4 세로
└─ gallery/
   ├─ 01.jpg          ← 슬라이드용 사진. 권장 비율 1:1 정사각
   ├─ 02.jpg
   └─ ... 09.jpg
```

## 교체 방법

1. 위 이름 그대로 사진을 넣으면 끝입니다. (경로는 `config.js` 의 `images` 에 정의)
2. 사진 개수를 바꾸려면 `config.js` 의 `images.gallery` 배열을 수정하세요.

## 최적화 권장

- 가능하면 **WebP** 로 변환해 같은 이름(`01.webp`)으로 두고 `config.js` 경로도 `.webp` 로 바꾸세요. 용량이 크게 줄어 모바일 로딩이 빨라집니다.
- 대문 사진은 가로 1200px, 갤러리 사진은 가로 1000px 내외로 리사이즈하면 충분합니다.
- 변환 예시 (cwebp 설치 시):
  ```bash
  cwebp -q 82 01.jpg -o 01.webp
  ```
